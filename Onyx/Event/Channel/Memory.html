    class Onyx::Event::Channel::Memory

    • Onyx::Event::Channel::Memory
    • Onyx::Event::Channel
    • Reference
    • Object

    Overview

    A non-blocking in-memory Event::Channel implementation. It relies on Crystal ::Channels.

    require "onyx-event/channel/memory"

# Subscriptions routine is spawned immediately
channel = Onyx::Event::Channel::Memory.new

channel.subscribe(MyEvent) do |event|
  puts "Got #{event}"
end

# Don't forget to sleep, otherwise the process exits
sleep
