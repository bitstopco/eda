    Overview

    An abstract event channel.

    According to Wikipedia:

    An event channel is a mechanism of propagating the information collected from an event generator to the event engine ... The events are stored in a queue, waiting to be processed later by the event processing engine.

    Currently implemented channels: Channel::Redis and Channel::Memory.

    A channel usually begins its routines upon initialization in separate fibers, thus being non-blocking.
